# cluster.tidb.ticat
A [ticat](https://github.com/innerr/ticat) modules repo: TiDB cluster management


## Apply this repo by running:
```bash
$> ticat hub.add innerr/cluster.tidb.ticat
```

## Tags
```bash
$> ticat - @ready       # out-of-the-box commands
$> ticat - @config      # configurer
```
